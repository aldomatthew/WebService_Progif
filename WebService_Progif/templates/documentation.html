METHODS

METHODS URL Description
GET /daftarproyek Mengambil semua data proyek yang ada
GET /daftarproyek/Sektor/<string:Sektor> Mengambil data proyek berdasarkan SEKTOR
GET /daftarproyek/Pulau/<string:Pulau> Mengambil data proyek berdasarkan PULAU
GET /daftarproyek/SektorPulau/<string:Sektor>/<string:Pulau> Mengambil data proyek berdasarkan SEKTOR,PULAU
GET /daftarproyek/SektorLokasi/<string:Sektor>/<string:Lokasi> Mengambil data proyek berdasarkan SEKTOR,LOKASI
GET /daftarproyek/PulauNilaiProyek/<string:Pulau>/<string:Nilai_Proyek> Mengambil data proyek berdasarkan PULAU,NILAI PROYEK
GET /daftarproyek/LokasiNilaiProyek/<string:Lokasi>/<string:Nilai_Proyek> Mengambil data proyek berdasarkan LOKASI, NILAI PROYEK
GET /daftarproyek/SektorNilaiProyek<string:Sektor>/<string:Nilai_Proyek> Mengambil data proyek berdasarkan SEKTOR, NILAI PROYEK
METHODS URL Description
POST /daftarproyek/isiproyek Menambahkan satu daftar proyek
PUT /daftarproyek/baru/<string:NamaProyek> Melakukan update berdasarkan NAMA PROYEK yang diinginkan
METHODS URL Description
DELETE /daftarproyek/deleteLokasi/<string:Lokasi> Melakukan request untuk menghapus data proyek berdasarkan Lokasi
DELETE /daftarproyek/deleteSektor/<string:Sektor> Melakukan request untuk menghapus data proyek berdasarkan Sektor
DELETE /daftarproyek/deleteSektorLokasi/<string:Sektor>/<string:Lokasi> Melakukan request untuk menghapus data proyek berdasarkan SEKTOR,LOKASI
DELETE /daftarproyek/deleteNamaProyek/<string:NamaProyek> Melakukan request untuk menghapus data proyek berdasarkan NAMA PROYEK
